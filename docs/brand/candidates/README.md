# Candidatos vetoriais não aprovados

Estes SVGs foram reconstruídos a partir dos PNGs provisórios e das pranchas de referência.

- não são ativos finais ou aprovados;
- não devem ser movidos para `public/brand` sem validação formal;
- o produto continua usando os PNGs públicos existentes;
- texto, proporções, curvas e cores devem ser comparados com os arquivos vetoriais originais quando estiverem disponíveis.

## Resultado da comparação visual

**Status:** rejeitados para uso público.

A comparação com os PNGs provisórios identificou diferenças relevantes na geometria do símbolo, espessura dos traços, proporções e composição tipográfica do logo horizontal. Os arquivos permanecem nesta pasta apenas como registro exploratório e não devem ser tratados como substitutos da marca aprovada.
